Использование:

"say-hi" -> "hi"

В данном примере после команды /say-hi пользователь получает в ответ сообщение с текстом hi.

  • Telegram
  • Discord
  • Whatsapp
  • Viber
"say {text}" -> "{text}"

Этот пример схож с предыдущим, однако теперь бот отправляет сообщение с текстом из исходного сообщения.
То есть при написании команды /say my name бот отправит текст my name.

  • Telegram
  • Discord
  • Viber
//"say {text}" -> "{text}"

Любой код можно комментировать с помощью //, а также /*code*/

  • Telegram
  • Discord
  • Whatsapp
  • Viber
"say {text}" -> mail("{text}", "e-mail")

В данном примере после команды /say-hi сообщение пересылается на mail по заданному адресу.

  • Telegram
  • Discord
  • Whatsapp
