Использование

BotCreator - инструмент, с помощью которого можно легко создавать простых ботов для любого мессенджера: discord, telegram, и тд.
Выгода такого подхода заключается в том, что всегда можно изменить тип месседжера, но логику переписывать не придется.

Код из себя представляет словарь ответов на различные сообщения пользователей:

  • "say-hi" -> "hi"

    В данном примере после команды /say-hi пользователь получает в ответ сообщение с текстом hi.

  • "say {text}" -> "{text}"

    Этот пример схож с предыдущим, однако теперь бот отправляет сообщение с текстом из исходного сообщения.
    То есть при написании команды /say my name бот отправит текст my name.

  • Префикс для команд можно поставить в настройках, дефолтный префикс - /

  • Код можно комментировать с помощью //
